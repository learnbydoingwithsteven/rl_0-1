📚 Theory

The RL Interaction Loop

At each time step t, the agent-environment interaction follows this cycle:

Interaction Cycle:
  1. Agent observes state st
  2. Agent selects action at
  3. Environment transitions to st+1
  4. Agent receives reward rt+1
Trajectory: s0, a0, r1, s1, a1, r2, s2, ...

Key Concepts:

  • State (s): Complete description of the environment
  • Action (a): Choice made by the agent
  • Reward (r): Scalar feedback signal
  • Transition: How environment changes

💻 Implementation

// Agent-Environment Interaction

class Agent {
    selectAction(state) {
        // Agent decision making
        return Math.random() > 0.5 ? 'left' : 'right';
    }
}

class Environment {
    constructor() {
        this.state = 0;
    }
    
    step(action) {
        // Environment dynamics
        const reward = action === 'right' ? 1 : -0.5;
        this.state = (this.state + 1) % 10;
        return { state: this.state, reward: reward };
    }
}

// Interaction loop
const agent = new Agent();
const env = new Environment();

for (let t = 0; t < 100; t++) {
    const action = agent.selectAction(env.state);
    const { state, reward } = env.step(action);
    console.log(`t=${t}: s=${state}, a=${action}, r=${reward}`);
}
