## 🎯 What Each Course Includes

Every single course (1-100) has:

### ✅ Individual HTML Application (`index.html`)
- **Header** with breadcrumb navigation
- **Theory Section** with explanations and learning objectives
- **Interactive Controls** with adjustable parameters:
  - Episodes slider (10-500)
  - Learning Rate slider (0.01-1.0)
  - Discount Factor (γ) slider (0-1)
  - Start/Reset buttons
- **Progress Bar** showing real-time completion
- **Navigation** to previous/next courses

### ✅ Individual JavaScript Demo (`script.js`)
- **Algorithm Implementation** specific to the course topic
- **Real-time Metrics Display**:
  - Episodes completed
  - Total reward
  - Average reward
  - Recent average (last 10 episodes)
  - Success rate
- **Interactive Visualizations** using Plotly.js:
  - Main chart: Reward per episode with moving average
  - Secondary chart: Cumulative reward over time
- **Event Handlers** for all controls
- **Data Management** with reset functionality

### ✅ Additional Features
- **Code Examples** showing algorithm implementation
- **3 Exercises** per course for practice
- **Responsive Design** works on all devices
- **Consistent Styling** using shared CSS
- **Utility Functions** from shared utils.js

6. **No Installation** - Runs directly in browser
7. **Offline Capable** - Only Plotly.js requires CDN

## 🚀 How to Use

### Quick Start
1. Open `index.html` in the root directory
2. Browse courses by level or use filters
3. Click any course to start learning
4. Adjust parameters and run interactive demos
5. Navigate between courses using Previous/Next buttons

### Learning Path
- **Beginners**: Start with Course 1, follow sequentially through Course 30
- **Intermediate**: Courses 31-60 build on beginner concepts
- **Advanced**: Courses 61-85 cover deep RL and modern algorithms
- **Expert**: Courses 86-100 explore cutting-edge research topics

### For Developers
- Each course is self-contained in its own folder
- Shared resources in `shared/` directory
- Easy to modify or extend individual courses
- Template available in `template/` directory
