# 100 Reinforcement Learning Courses: Zero to Hero

A comprehensive curriculum covering reinforcement learning from absolute basics to cutting-edge research topics. Each course includes interactive visualizations, numerical analysis, and graphical demonstrations.

## Curriculum Structure

### 🟢 Beginner Level (Courses 1-30)
Foundation concepts, multi-armed bandits, MDPs, and dynamic programming

### 🟡 Intermediate Level (Courses 31-60)
Temporal difference learning, Q-learning, SARSA, function approximation

### 🟠 Advanced Level (Courses 61-85)
Deep Q-Networks, policy gradients, actor-critic methods, PPO, A3C

### 🔴 Expert Level (Courses 86-100)
Multi-agent RL, meta-learning, model-based RL, advanced research topics

## Features

Each course includes:
- **Interactive Visualizations**: Real-time algorithm demonstrations
- **Numerical Analysis**: Step-by-step calculations and metrics
- **Graphical Results**: Charts, heatmaps, and performance plots
- **Code Examples**: Fully functional implementations
- **Theory Explanations**: Mathematical foundations and intuitions

## Quick Start

1. Open `index.html` in the root directory
2. Navigate through courses sequentially or jump to specific topics
3. Each course runs independently in your browser
4. No installation required - pure HTML/CSS/JavaScript

## Course Navigation

- **Sequential Learning**: Follow courses 1→100 for complete mastery
- **Topic-Based**: Jump to specific topics using the course index
- **Difficulty-Based**: Filter by beginner/intermediate/advanced/expert
- **Search**: Find courses by keywords or concepts

## Technologies Used

- Pure JavaScript (no frameworks required)
- Plotly.js for interactive charts
- D3.js for advanced visualizations
- Canvas API for custom animations
- Web Workers for performance

## Learning Path

1. **Weeks 1-4**: Beginner courses (1-30)
2. **Weeks 5-8**: Intermediate courses (31-60)
3. **Weeks 9-12**: Advanced courses (61-85)
4. **Weeks 13-16**: Expert courses (86-100)

Total estimated time: 16 weeks of dedicated study
