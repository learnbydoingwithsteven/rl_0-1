📚 Theory

What is Reinforcement Learning?

Reinforcement Learning (RL) is a type of machine learning where an agent learns to make decisions by interacting with an environment. The agent receives rewards or penalties based on its actions and learns to maximize cumulative rewards over time.

Key Concept: Unlike supervised learning (where we have labeled data) or unsupervised learning (where we find patterns), RL learns through trial and error by receiving feedback from the environment.

Core Components:

  • Agent: The learner or decision maker
  • Environment: The world the agent interacts with
  • State (s): Current situation of the agent
  • Action (a): Choices available to the agent
  • Reward (r): Feedback signal from the environment
  • Policy (π): Strategy that maps states to actions
RL Goal: Maximize cumulative reward G = Σ rt

The RL Loop:

  1. Agent observes current state st
  2. Agent selects action at based on policy π
  3. Environment transitions to new state st+1
  4. Agent receives reward rt+1
  5. Repeat

💻 Implementation


// Simple RL Agent
class SimpleRLAgent {
    constructor() {
        this.totalReward = 0;
        this.episodeRewards = [];
    }
    
    // Agent takes action in environment
    takeAction(state) {
        // Random action for now (exploration)
        return Math.random() > 0.5 ? 'left' : 'right';
    }
    
    // Agent receives reward and learns
    receiveReward(reward) {
        this.totalReward += reward;
        this.episodeRewards.push(reward);
    }
    
    // Calculate average reward
    getAverageReward() {
        if (this.episodeRewards.length === 0) return 0;
        return this.totalReward / this.episodeRewards.length;
    }
}

// Environment
class SimpleEnvironment {
    constructor() {
        this.state = 'start';
    }
    
    // Environment responds to agent's action
    step(action) {
        // Simple reward structure
        const reward = action === 'right' ? 1 : -1;
        this.state = action === 'right' ? 'goal' : 'start';
        return { state: this.state, reward: reward };
    }
}
